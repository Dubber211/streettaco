# StreetTaco

Community-powered food truck finder. Spot a truck, drop a pin, and help your community eat well.

**Live:** [streettaco.vercel.app](https://streettaco.vercel.app)

## Stack

- React + Vite
- Supabase (database + auth)
- Leaflet (maps)
- Vercel (hosting)

## Local Development

```bash
npm install
npm run dev
```

Requires a `.env` file with:

```
VITE_SUPABASE_URL=your-supabase-url
VITE_SUPABASE_ANON_KEY=your-supabase-anon-key
```
